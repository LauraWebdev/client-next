# SpinShare - Client3
A modern cross-platform desktop client for SpinShare

## Overview
- [Setup](#setup)
- [Building](#building)

### Setup
#### Installing all dependencies
```sh
cd UserInterface && npm install
```

#### Running a dev server
```sh
cd UserInterface && npm run dev
```

#### Starting the client
```sh
dotnet run
```

### Building
#### Building the UI
```sh
cd UserInterface && npm run build
```
